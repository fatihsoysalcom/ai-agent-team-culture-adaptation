# AI Agent Team Culture Adaptation

This example demonstrates how an AI agent can adapt its communication style and content based on a predefined 'team culture profile'. It simulates two different AI agents, each configured with distinct cultural attributes like tone, core values, and humor level, showcasing how these influence their responses to user queries. This illustrates the concept of teaching or configuring AI with non-technical team norms.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
